## Setup

### NodeJS Install

#### Ubuntu

```sh
curl -sL https://deb.nodesource.com/setup_10.x | sudo -E bash -
sudo apt-get install -y nodejs
```
#### OSX

```sh
brew update
brew install node
```

### Install gems

```sh
npm install
```
